Elementos de un Prompt

Instrucción: Una tarea especifica o instruccion.

Contexto: Información externa que puede orientar el moddelo a una mejor respuesta.

Datos de entrada: la entrada o pregunta a la que queremos encrontrarle respuesta.

Indicador de salida: la forma en la que queremos que nos responda el modelo.

Para escribir un prompt no se necesitan estos 4 elementos en el 100% de las ocasiones, todo depende del contexto.

Ejemplo de cada elemento

Eres un academico de renombre en las ciencias politicas (Contexto) Realiza una comparación sobre (Introducción) las ideas de las diferentes corrientes anarquistas (anarcocapitalismo, anarcocomunismo, panarquismo) (input) En formato de un cuadro comparativo (output)

Tips generales para diseñar prompts

Empezar simple

Empezar con prompts simples e irle agregando mas elementos y contexto a medida que busques mejores resultados. Cuando tienes una tarea muy grnade puedes dividirla en pequeñas tareas y así mantener tus prompts simples.

La instrucción

Usa comandos de instrucción como "Escribe", "Clasifica", "Resume", "Traduce", "Ordena", etc.

Algunos recomendan poner las instrucciones al principio del prompt, otra recomendación es usar separadores como "###", por ejemplo:

### Instrucción ###

Traduce siguiente texto a ingles:

Texto: "¡Hola!"

Especificidad

Se muy especifico en lo que quieres que el modelo haga, mientras mas descriptivo y detallado el prompt es, mejores son los resultados. Los detalles que des deben ser relevantes, incluir muchos detalles innecesarios no es la mejor practica, Por ejemplo:

Extraer el nombre de los lugares en el siguiente texto.

Formato deseado:

Lugar: {lista_separada_por_comas_de_nombres_de_empresa}

Entrada: "Aunque estos avances son alentadores para los investigadores, muchas cosas siguen siendo un misterio. "A menudo tenemos una caja negra entre el cerebro y el efecto que vemos en la periferia", dice Henrique Veiga-Fernandes, neuroinmunólogo del Centro Champalimaud para lo Desconocido de Lisboa. "Si queremos utilizarlo en el contexto terapéutico, tenemos que entender realmente el mecanismo.""

Evita Imprecisiones

Dados los anteriores tips es facil caer en la trampa de querer hacerse el inteligente y potencialmente crear descripciones imprecisas, normalmente lo mejor es ser especifico y directo. Es como en la comunicación, mientras mas directo seas mas efectivamente se transmite el mensaje.
