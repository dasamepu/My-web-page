Elementos de un Prompt

Instrucción: Una tarea especifica o instruccion.

Contexto: Información externa que puede orientar el moddelo a una mejor respuesta.

Datos de entrada: la entrada o pregunta a la que queremos encrontrarle respuesta.

Indicador de salida: la forma en la que queremos que nos responda el modelo.

Para escribir un prompt no se necesitan estos 4 elementos en el 100% de las ocasiones, todo depende del contexto.

Ejemplo de cada elemento

Eres un academico de renombre en las ciencias politicas (Contexto) Realiza una comparación sobre (Introducción) las ideas de las diferentes corrientes anarquistas (anarcocapitalismo, anarcocomunismo, panarquismo) (input) En formato de un cuadro comparativo (output)

Tips generales para diseñar prompts

Empezar simple

Empezar con prompts simples e irle agregando mas elementos y contexto a medida que busques mejores resultados. Cuando tienes una tarea muy grnade puedes dividirla en pequeñas tareas y así mantener tus prompts simples.

La instrucción

Usa comandos de instrucción como "Escribe", "Clasifica", "Resume", "Traduce", "Ordena", etc.

Algunos recomendan poner las instrucciones al principio del prompt, otra recomendación es usar separadores como "###", por ejemplo:

### Instrucción ### Traduce siguiente texto a ingles: Texto: "¡Hola!"
